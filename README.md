# 班级优化大师 - 宠物养成版 🎓🐾

一个有趣且互动性强的课堂管理工具，类似班级优化大师，但融入了宠物养成元素，让学生在上课时积极参与回答问题。

> ⚠️ **注意**：这不是官方的"班级优化大师"，而是一个独立开发的趣味版本。

---

## 📅 版本信息

- **当前版本**：1.0
- **发布日期**：2026 年 4 月 10 日

---

## ✨ 核心功能

### 🎮 学生管理系统
- **宠物头像系统**：每个学生都有独特的宠物头像
- **等级系统**：学生通过积累经验值升级，最高 10 级
  - 满级时有特殊彩虹徽章特效
  - 经验条实时显示升级进度

### 🏆 积分系统
- **加分/减分界面**：便捷的积分操作面板
- **随机抽选**：公平随机选择学生回答问题
- **积分排行榜**：激励学生积极参与

### 🛒 兑换系统
- 学生可以使用积分兑换奖励
- 增强学习动力和参与感

### 🔐 用户系统
- **登录认证**：保护数据安全
- **角色权限**：区分管理员和普通用户
- **注册功能**：即将开放全员注册

---

## 🛠️ 技术栈

- **前端**：HTML5, CSS3, JavaScript (原生)
- **后端服务**：Supabase (数据库 + 认证)
- **部署**：静态网站托管
- **图标**：内联 SVG / Emoji

---

## 🚀 快速开始

### 1. 克隆项目
```bash
git clone <repository-url>
cd <project-directory>
```

### 2. 配置 Supabase
在 `index.html` 中配置您的 Supabase 凭证：
```javascript
const supabase = createClient('YOUR_SUPABASE_URL', 'YOUR_SUPABASE_ANON_KEY');
```

### 3. 创建数据库表
参考项目文档创建所需的数据表结构。

### 4. 运行
直接在浏览器中打开 `index.html` 即可使用。
